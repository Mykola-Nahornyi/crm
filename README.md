# B2B Contracts & Billing Platform

RESTful backend application for managing clients, contracts, invoices and payments.

## Tech Stack
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Docker
- OpenAPI (Swagger)

## Local Development

### Requirements
- Java 17
- Docker & Docker Compose

### Run PostgreSQL
```bash
docker compose up -d

### Run APP on Windows
```bash
.\mvnw spring-boot:run
